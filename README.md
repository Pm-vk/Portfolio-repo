# Prashant Mishra - 3D Portfolio

A high-performance, immersive 3D portfolio experience built for **Prashant Mishra**. This platform showcases a "Cyber-Premium" design aesthetic, utilizing modern web technologies to present a professional narrative that stands distinct from a traditional resume.

## Live Demo
🚀 **[View Live Portfolio](https://prashant-portfolio13.netlify.app/)**

## ✨ Features
- **Immersive 3D Character**: Interactive scene with real-time lighting and custom thematic materials.
- **Micro-Animations**: GSAP-powered scroll triggers and smooth transitions.
- **Glassmorphism UI**: High-contrast, frosted-glass components with neon accents.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewports.
- **Dynamic Content**: Narrative-driven project showcase and career timeline.

## 🛠 Tech Stack
- **Frontend**: React.js, TypeScript, Vite
- **3D Engine**: Three.js, React Three Fiber, Drei, Cannon
- **Animation**: GSAP (ScrollTrigger, ScrollSmoother)
- **Styling**: Vanilla CSS, Tailwind CSS
- **Deployment**: Netlify

## 📂 Project Structure
- `src/components/Character`: 3D scene and character logic.
- `src/components/styles`: Centralized CSS design system.
- `src/context`: Loading and animation state management.
- `public/images`: Generated high-fidelity project mockups.

## 🚀 Getting Started
1. Clone the repository.
2. Run `npm install` to set up dependencies.
3. Run `npm run dev` to start the local development server.
4. Run `npm run build` to generate the production bundle.
